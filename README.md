# DevOps Portfolio

🔗 Live Demo: https://portfolio-omega-neon-37.vercel.app

## Description
Personal portfolio built with React to showcase my projects, skills, and certifications.

This project also demonstrates a simple DevOps workflow using CI/CD.

## Stack
- React
- Vite
- GitHub Actions (CI)
- Vercel (Deployment)

## DevOps Workflow
1. Code is pushed to GitHub
2. GitHub Actions runs the CI pipeline
3. If successful, Vercel automatically deploys the site

Developer → Git Push → GitHub Actions → Vercel Deploy
